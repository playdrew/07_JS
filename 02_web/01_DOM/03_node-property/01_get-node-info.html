01. 요소 노드의 텍스트 조작

nodeValue

        nodeValue 라는 프로퍼티는
        getter, setter 가 모두 존재하는 프로퍼티이다.
        nodeValue 프로퍼티 참조 시 노드 객체의 값
        (텍스트 노드의 텍스트)을 반환해준다.
        여기서 주의할 점은 텍스트가 아닌
        요소 노드인 경우에는 null 을 반환한다.
변경 전

textContent

        textContent 프로퍼티도 setter 와 getter 가
        모두 존재하는 프로퍼티다.
        하지만 위에서 nodeValue 와 다른 점은
        요소 노드의 텍스트와 모든 자손 노드의
        텍스트를 모두 취득하거나 변경할 수 있다.
