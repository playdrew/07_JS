05.HTMLCollection 과 NodeList

        HTMLCollection 과 NodeList 는
        여러 개의 결과 값을 담기 위한
        DOM 걸렉션 객체의 종류이다.
        이 두 가지는 모두 유사 배열 객체이면서
        이터러블(iterator)이다.
        노드 객체의 상태 변화를 실시간으로 반영하는
        살아있는 DOM 컬렉션 객체이다.

        컬렉션 객체이다.
