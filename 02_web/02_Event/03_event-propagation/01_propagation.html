01. 이벤트의 전파

        생성 된 이벤트 객체는 이벤트를 발생 시킨
        DOM 요소인 이벤트 target 을 중심으로 DOM 트리를
        통해 전파된다.
        1. 캡쳐링 단체(capturing phase) : 상위 요소에서 하위요소로 전파된다
        2. 타깃 단계(target phase) : 이벤트가 이벤트 타겟에 도달
        3. 버블링 단계(bubbing phase) : 이벤트가 하위요소에서 상위요소로 전파

        캡쳐링 단계의 이벤트도 캐치할 수 있다.
